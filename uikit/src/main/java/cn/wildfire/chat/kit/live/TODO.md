UI 优化，要求美观专业
1. 主播端和观众端都需要支持悬浮窗，可参考SingleCallActivity 实现
2. 视频要占满整个屏幕，沉浸式体验
3. 主播端和观众端
   1. 左上角显示主播信息，包括头像和昵称等
   2. 右上角显示分享、悬浮、关闭按钮
      1. 分享按钮，相当于转发直播开始按钮，可以参考转发功能实现，有现成的转发界面等
      2. 悬浮按钮，缩小到悬浮窗
      3. 关闭按钮，主播结束直播，观众，停止观看直播
   3. 底部左侧显示 说点什么输入框
4. 主播端
   1. 底部说点什么右侧显示，切换摄像头，连麦管理按钮
      1. 切换摄像头，切换前后摄像头
      2. 连麦管理，采用 bottomsheet 方式弹出连麦管理界面
         1. 当前正在连麦的观众，支持结束连麦
         2. 申请连麦的观众，支持接通连麦
         3. 可邀请的观众，支持邀请连麦
5. 观众端
   1. 底部有点什么 右侧显示 连麦按钮
      1. 点击后，bottomsheet 方式显示连麦方式：视频连麦，语音连麦
